给朋友的使用说明
==================

一、先做什么
1. 先把收到的压缩包解压。
2. 进入解压后的文件夹。
3. 双击“双击运行程序.bat”。

二、如果能正常打开
程序窗口名称是“传媒资讯交互数据分析平台”。
可以这样演示：
1. 左侧选择“题材分类”或“发布渠道”，点击“查询数据”。
2. 右边看数据表格和上方指标卡。
3. 切换“图表类型”，点击“生成分析图表”。
4. 打开“示例场景”标签页，点击不同例子，会自动套用筛选条件。
5. 点击左侧“绘制传媒主题标识”，会弹出 Turtle 绘图窗口。
6. 点击“查看结论建议”，会弹出分析结论。

三、如果打不开
可能是电脑没安装 Python。
解决方法：
1. 打开 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本。
3. 安装时一定勾选“Add Python to PATH”。
4. 安装完成后，重新双击“双击运行程序.bat”。

四、如果提示缺少 Matplotlib
正常情况下 bat 文件会自动安装。
如果自动安装失败，可以在文件夹空白处按住 Shift + 右键，选择“在终端中打开”，输入：

python -m pip install matplotlib

安装完成后，再双击“双击运行程序.bat”。

五、需要提交哪些文件
如果是交作业，至少提交：
1. media_insight_platform.py
2. Python期末作品设计报告_传媒资讯交互数据分析平台.docx

如果是发给别人运行，建议整个文件夹压缩后一起发送。
