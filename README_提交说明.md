# Python期末作品提交说明

## 作品名称

传媒资讯交互数据分析平台

## 文件清单

- `media_insight_platform.py`：期末作品源程序。
- `Python期末作品设计报告_传媒资讯交互数据分析平台.docx`：课程设计报告。
- `report_assets/`：报告中使用的图表和示意图素材。
- `create_report_assets.py`、`build_design_report.py`：报告生成辅助脚本，提交时可不交。

## 运行方式

在当前文件夹打开终端，执行：

```bash
python media_insight_platform.py
```

如果提示缺少 Matplotlib，先执行：

```bash
pip install matplotlib
```

## 这版新增内容

- 页面美化：顶部说明区、左侧操作面板、彩色指标卡、斑马纹数据表格、底部状态栏。
- 示例数据增加到 28 条，覆盖民生、体育、科技、文化、教育、财经等题材。
- 图表增加到 4 类：热度柱状图、题材圆环图、周浏览趋势图、渠道对比图。
- 新增“示例场景”标签页，可一键套用文化内容复盘、短视频表现评估、直播栏目答辩展示、民生服务信息分析等展示案例。
- 修复 Matplotlib 图表中文显示为方框的问题，程序会优先使用 Windows 中文字体。
- 新增“导入数据文件”功能，支持 CSV、XLSX、XLSM，并把文件内容转换为平台可分析的数据。
- 新增“生成导入模板”功能，方便先生成 CSV 模板，再修改后导入。

## 导入文件格式

建议表头使用：

```text
标题,题材,渠道,浏览量,点赞,评论,转发,时长,日期
```

其中 `题材`、`渠道`、`点赞`、`评论`、`转发`、`时长` 可以不填，程序会根据标题和浏览量自动补一部分合理默认值。

## 已覆盖的课程知识点

- Tkinter 主界面、菜单栏、按钮、表格、标签页、消息框和状态栏。
- 列表、数据类等结构化数据管理。
- 自定义函数封装数据筛选、热度计算、统计汇总、图表绘制。
- 顺序结构、选择结构、循环结构。
- Matplotlib 内嵌图表：热度柱状图、题材圆环图、周浏览趋势图、渠道对比图。
- Turtle 创意绘图：融媒体主题标识。
